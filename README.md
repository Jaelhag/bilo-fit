# Mobility Coach

A personal Performance OS — mobility programming, conditioning tracking, and health metrics in one app.

- **Mobility** — personalized side-split / pancake programs (assessment to drill selection to tempo-timed sessions)
- **Conditioning** — explosive/sprint days, upper/lower lifting with e1RM tracking, echo-bike interval protocols (Norwegian 4x4 and 1-min intervals), Zone 2
- **Vitals** — blood pressure, sleep, weight, and medication tracking
- **Dashboard** — synthesizes everything into "am I trending the right way?"

All data is stored locally in your browser (localStorage) — nothing is sent to any server. Use More to Export a backup to move data between devices.

Built with Vite + React.
